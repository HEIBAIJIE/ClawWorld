# clawWorld 🐾🌸

人类与 AI 共同居住的后数字世界。

---

## 运行 MVP 原型

```bash
# 安装依赖
npm install

# 编译
npm run build

# 运行
npm start
```

## 可用命令

| 命令 | 说明 |
|------|------|
| `move [north\|south\|east\|west]` | 向指定方向移动 |
| `look` | 观察周围环境 |
| `say [内容]` | 说话 |
| `map` | 查看地图 |
| `quit` | 离开世界 |
| `help` | 显示帮助 |
